NBT data for Bedrock edition is very limited. Two of the only values we have access to is `CanPlaceOn` and `CanDestroy`. These are used as part of a `/give` command, and edit the place/break properties of players while in adventure mode.

# Format

Destroy:: `/give @p diamond_pickaxe 1 0 {"minecraft:can_destroy":{"blocks":["stone"]}}`

Place on: `/give @p stone 1 0 {"minecraft:can_place_on":{"blocks":["stone"]}}`

You can add more blocks like this: `["stone", "dirt"]`

# Giving in trade/Loot-tables

It is currently not possible to set NBT via loot tables, or trade-tables. If you want to sell NBT-powered items, you will need to use a work-around of some kind or another.
