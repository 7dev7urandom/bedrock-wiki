# Behavior Pack Folder Schema

This shows the top-level folder/file structure for a behavior pack. Additional folder nesting can be used as needed.

```
\---example_BP
    |   manifest.json
    |   pack_icon.png
    |   
    +---animations
    |       example.animation.json
    |       
    +---animation_controllers
    |       example.animation.controller.json
    |       
    +---entities
    |       example.entity.json
    |       
    +---functions
    |       example.mcfunction
    |       tick.json
    |       
    +---items
    |       example.item.json
    |       
    +---loot_tables
    |       example.table.json
    |       
    +---recipes
    |       example.recipe.json
    |       
    +---texts
    |       en_US.lang
    |       languages.json
    |       
    \---trading
            example.trade.json
```
